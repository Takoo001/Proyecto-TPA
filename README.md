# SIMO CO2 - Sistema de Monitoreo de Huella de Carbono 
Este proyecto es una plataforma de visualización industrial diseñada para monitorear el consumo energético y la huella de carbono en plantas de producción. El sistema procesa datos en tiempo real de sensores simulados y presenta métricas clave mediante un panel interactivo.

## Guía de Inicio Rápido

### Requisitos Previos
* **Node.js** (Versión LTS recomendada)
* **npm** (Viene incluido con Node.js)

### Instalación de Dependencias
